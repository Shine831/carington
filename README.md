# 🛡️ E-JARNALUD SOFT — Elite IT Solutions

> **Architecture de Confiance.** Une plateforme Next.js ultra-performante dédiée à la cybersécurité et à l'infrastructure IT, conçue avec un système de design "Elite" Rouge & Blanc.

## ✨ Caractéristiques Elite

- **💎 Design "Écarlate" v3** : Une esthétique minimaliste mais percutante utilisant une palette Rouge Cramoisi, Blanc Pur et Anthracite.
- **🐭 Curseur Interactif** : Un curseur magnétique personnalisé avec anneau réactif pour une navigation immersive.
- **🍱 Bento Grid Services** : Une organisation moderne et structurée des prestations (Audit, Réseau, Infogérance).
- **🌪️ Animations Fluides** : Propulsé par **Framer Motion** avec des transitions de page `@AnimatePresence` et des révélations au scroll.
- **📐 3D Tilt Cards** : Tableaux de bord interactifs avec effet d'inclinaison 3D suivant le mouvement de la souris.
- **🌫️ Texture Grain/Noise** : Superposition de bruit organique pour une sensation tactile et un rendu haut de gamme.

## 🛠️ Stack Technique

- **Framework** : [Next.js 15+](https://nextjs.org/) (App Router)
- **Stylisation** : [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations** : [Framer Motion](https://www.framer.com/motion/)
- **Iconographie** : [Lucide React](https://lucide.dev/)
- **Typographie** : Inter & Serif Italic (Google Fonts)

## 🚀 Installation & Développement

1. **Cloner le projet** :
   ```bash
   git clone https://github.com/votre-username/carington.git
   cd carington
   ```

2. **Installer les dépendances** :
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement** :
   ```bash
   npm run dev
   ```
   Ouvrez [http://localhost:3000](http://localhost:3000) pour voir le résultat.

## 🛰️ Déploiement sur Vercel

Pour déployer sur Vercel (recommandé) :

1. Poussez votre code sur GitHub.
2. Connectez votre compte GitHub sur [Vercel](https://vercel.com).
3. Importez le projet `carington`.
4. Vercel détectera la configuration Next.js. Cliquez sur **Deploy**.
